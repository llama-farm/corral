# Corral Skill

## Description
Add authentication, billing, and feature gating to any app.

## Usage
When the user asks to add auth, login, billing, payments, or feature gating to an app:
1. Check if `corral.yaml` exists (already initialized)
2. If not: run `npx create-corral init`
3. Read `CORRAL.md` for project-specific context
4. Use `npx create-corral add` commands as needed
5. Run `npx create-corral doctor` to verify
